Portfolio

Here are some of my projects.

battleship game

I created a Battleship game in which users play against an enemy AI (Artificial Intelligence). Users can place their own battleships on a GUI (Graphical User Interface) that represents the game board and is updated in real time. It includes user instructions, a constantly updated status display, and a leaderboard that keeps track of users with the highest scores. Built the game using Java and Swing, and incorporates complex search, 2D arrays, parsing and file I/O, and J-Unit testing.

photosort

I am currently developing a photo organization app, written in Java, to help users organize shopping lists, laundry, etc. Users can select photos from gallery or use the camera from within the app. Planning to implement user accounts, cloud storage of user data and photos, and photo cropping.
